About MRC

MRC is a simulated version of the FRC game Infinite Recharge inside of Minecraft. Thanks to a custom-coded plugin and a resource pack, you can play Infinite Recharge on Minecraft Java Edition or Bedrock Edition in 3v3 matches with your friends!

The original Minecraft Robotics Competition Regional Event was held on December 19, 2020. Check out the full event video and the MRC trailer embedded below.

MRC is open source. Help improve it!

Server Status: Offline

Since MRC has not received any updates in a few years, we do not currently host an MRC server at this time. If this changes, we will update this page.
